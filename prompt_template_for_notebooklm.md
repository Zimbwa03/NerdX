# Prompt Template for Generating A Level Chemistry Notes in Google NotebookLM

## Instructions for Using This Prompt:

Copy the template below and replace:
- **[TOPIC NAME]** with the specific topic name (e.g., "Reaction Kinetics", "Group 2 Elements")
- **[TOPIC DESCRIPTION]** with a brief description of the topic
- **[CATEGORY]** with the category: Physical Chemistry, Inorganic Chemistry, Organic Chemistry, or Analysis
- **[LEVEL]** with AS Level or A2 Level

## ADDITIONAL TIPS:

1. If the topic has calculations (like Reaction Kinetics, Chemical Energetics), emphasize including:
   - Step-by-step worked examples
   - Formula derivations
   - Unit conversions
   - Common calculation pitfalls

2. If the topic has mechanisms (like Hydrocarbons, Benzene), emphasize:
   - Arrow-pushing mechanisms
   - Intermediate structures
   - Conditions for each step
   - Stereochemical considerations

3. If the topic has trends (like Group 2, Group 17, Periodic Table), emphasize:
   - Comparative tables
   - Explanations of trends
   - Exceptions and their reasons
   - Comparison across periods and groups

4. If the topic has applications (like Polymerisation, Analytical Techniques), emphasize:
   - Real-world examples
   - Industrial processes
   - Practical uses
   - Environmental impact

5. After generating the notes, you can add a follow-up request:
   "Please also provide a summary with 15-20 key points and 12-15 exam tips for this topic in bullet format."

## NOTES FOR CONSISTENCY:

- Always start with an introduction that sets context
- Use consistent section numbering (1.0, 2.0, 3.0, etc.)
- Include worked examples for calculation-heavy topics
- Add "Examiner's Tips" throughout
- End with a clear summary or conclusion
- Use proper chemical notation throughout
- Maintain the same level of detail and academic tone
