## Quick Reference Guide:

**For Physical Chemistry topics** (Reaction Kinetics, Chemical Energetics, etc.):
- Emphasize calculations, formulas, and worked examples
- Include graphical interpretations
- Explain theoretical models in detail

**For Inorganic Chemistry topics** (Group 2, Group 17, Periodic Table, etc.):
- Emphasize trends, patterns, and comparisons
- Include comparative tables
- Explain exceptions to trends
- Link properties to electronic structure

**For Organic Chemistry topics** (Hydrocarbons, Benzene, Phenols, etc.):
- Emphasize reaction mechanisms with arrow-pushing
- Include step-by-step reaction pathways
- Explain stereochemistry where relevant
- Cover synthesis applications

**For Analysis topics** (Analytical Techniques, etc.):
- Emphasize interpretation of data
- Include example spectra/readings
- Explain how to use techniques in structure determination
