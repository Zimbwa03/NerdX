## Prerequisites

- Node.js 18+
- npm or yarn
- Android Studio (for Android development)
- Xcode (for iOS development, Mac only)

## Installation

```bash
npm install
```

## Running the App

### Android
```bash
npm run android
```

### iOS
```bash
npm run ios
```

## Development

Start Metro bundler:
```bash
npm start
```
